Sobre o Moodify

Moodify é uma página web que identifica suas emoções a partir da expressão facial capturada pela câmera. Tudo é feito de forma leve e interativa diretamente no navegador.

A detecção é realizada com a biblioteca face-api.js, baseada em TensorFlow.js. Os modelos de aprendizado são carregados de um CDN e executados localmente, sem enviar suas imagens para servidores externos. Para acessar a câmera, utilizamos a API MediaDevices do navegador.

Conforme o humor predominante é identificado, a página mostra o emoji correspondente e altera a cor de fundo, junto de uma porcentagem de confiança. Você pode personalizar os humores modificando o código em index.html.
